# 📊 Sales Performance Analytics Dashboard

An end-to-end data analytics project that demonstrates how raw sales data can be transformed into meaningful business insights using Python, SQL, Excel, and Power BI.

---

## Project Overview

Businesses generate thousands of sales transactions every day. This project analyzes retail sales data to answer important business questions regarding revenue, profitability, customer behavior, and regional performance.

The project follows the complete analytics lifecycle:

- Data Collection
- Data Cleaning
- Exploratory Data Analysis (EDA)
- SQL Business Analysis
- Dashboard Development
- Business Recommendations

## Business Questions

This project answers questions such as:

- Which region generates the highest sales?
- Which products are most profitable?
- Which customers generate the most revenue?
- How do sales change month-to-month?
- Which categories should receive more investment?
